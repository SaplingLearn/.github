# Sapling Learn

**Sapling is an AI-powered study companion that adapts to how you learn.**

Upload your course materials, and Sapling builds a live knowledge graph of your understanding — then helps you close the gaps through adaptive tutoring, quizzes, and flashcards.

---

## What Sapling Does

| Feature | Description |
|---|---|
| **Knowledge Graph** | Visualizes mastery across concepts in real time. Updates after every session and quiz. |
| **Adaptive Tutoring** | Three modes: Socratic (guided questions), Expository (direct explanation), Teach-back (explain it back to the AI). |
| **Quizzes** | AI-generated quizzes that target your weakest concepts, scaling in difficulty as you improve. |
| **Flashcards** | Auto-generated per topic with spaced-repetition ratings (Easy / Hard / Forgot). |
| **Study Guides** | Exam-ready guides generated from your uploaded course materials. |
| **Syllabus & Calendar** | Paste a syllabus — Sapling extracts assignments, deadlines, and topics into a calendar view. |
| **Document Library** | Upload PDFs, notes, slides, or Word docs. Sapling extracts summaries, key takeaways, and flashcard topics. |
| **Study Rooms** | Invite classmates, compare knowledge graphs, and surface class-wide weak areas and misconceptions. |

## Tech Stack

**Frontend** — Next.js (TypeScript), React, Tailwind CSS, D3.js

**Backend** — FastAPI (Python), Google Gemini (AI), Google OAuth + Calendar API

**Database** — Supabase (PostgreSQL)

**Infrastructure** — Docker / Docker Compose
